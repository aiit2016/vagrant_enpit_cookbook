# Requirements

Installing Berkshelf:

```
bundle install --path=vendor/bundle --binstubs=vendor/bin
```

For Berkshelf:

```
$ vagrant plugin install vagrant-berkshelf
```

For the desired version of Chef:

```
$ vagrant plugin install vagrant-omnibus
```
